# Architecture

Thư mục này mô tả kiến trúc backend và quan hệ giữa các thành phần chính.

- `architecture.mmd`: sơ đồ Mermaid thể hiện luồng yêu cầu và thành phần (FastAPI, middlewares, Lifespan init, Redis, Postgres, LangChain agent và tools).

Mở `architecture.mmd` bằng trình xem hỗ trợ Mermaid (VS Code Mermaid Preview, MkDocs + mermaid plugin, v.v.).
